**Rule:** every function takes plain data, returns plain data. No fetch, no DB, no Date.now (pass timestamp as arg). Purely testable.

Example:
```ts
export function computeLine(input: LineInput, rate: MetalRate, tenantCfg: TenantConfig): LineComputed {
  const metalValue = round(input.netWeight * rate.ratePerGram, 2)
  const making =
    input.makingChargeType === 'flat' ? input.makingChargeValue
    : input.makingChargeType === 'per_gram' ? input.netWeight * input.makingChargeValue
    : (metalValue * input.makingChargeValue) / 100
  const wastage = (metalValue * input.wastagePercent) / 100
  const stoneValue = (input.stones ?? []).reduce((s, x) => s + x.weightCt * x.ratePerCt * x.count, 0)
  const lineTotalPreTax = metalValue + making + wastage + stoneValue - (input.lineDiscount ?? 0)
  return {
    metalValue,
    makingCharge: round(making, 2),
    wastageAmount: round(wastage, 2),
    stoneValue: round(stoneValue, 2),
    lineTotalPreTax: round(lineTotalPreTax, 2),
    taxAmount: round((lineTotalPreTax * input.taxRate) / 100, 2),
    lineTotal: round(lineTotalPreTax * (1 + input.taxRate / 100), 2),
  }
}
```

Uses **Decimal.js** internally for accuracy; converts to `number` at edges. Never math with JS float on money.
